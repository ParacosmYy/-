# 基于 ST89C52 的 SPWM 波形发生器

## 项目简介

本设计以 ST89C52（AT89C52 兼容）单片机为核心，实现 **1Hz~20Hz 可调 SPWM（正弦脉宽调制）波形发生器**。

主要功能：
- P3.7 输出 SPWM 波形，正弦包络由 100 点查表法生成
- K1/K2 按键调节输出频率（1Hz 步进，范围 1~20Hz）
- 两位共阳七段数码管实时显示当前频率值

## 硬件连接

### 引脚分配表

| 单片机引脚 | 功能 | 连接说明 |
|-----------|------|---------|
| P1.0 | 七段数码管 a 段 | 段选，低电平点亮 |
| P1.1 | 七段数码管 b 段 | 段选，低电平点亮 |
| P1.2 | 七段数码管 c 段 | 段选，低电平点亮 |
| P1.3 | 七段数码管 d 段 | 段选，低电平点亮 |
| P1.4 | 七段数码管 e 段 | 段选，低电平点亮 |
| P1.5 | 七段数码管 f 段 | 段选，低电平点亮 |
| P1.6 | 七段数码管 g 段 | 段选，低电平点亮 |
| P2.0 | 数码管十位位选 | 低电平选中十位数码管 |
| P2.1 | 数码管个位位选 | 低电平选中个位数码管 |
| P3.2 | K2 按键（频率-） | 按键一端接 P3.2，另一端接 GND，低电平有效 |
| P3.3 | K1 按键（频率+） | 按键一端接 P3.3，另一端接 GND，低电平有效 |
| P3.7 | SPWM 波形输出 | 接负载或示波器，必要时加驱动电路 |

### 硬件接线图

```
                          ST89C52
                       +-----------+
                       |           |
     +--[11.0592MHz]--+|XTAL1 XTAL2|+--[11.0592MHz]--+
     |  (晶振)        |           |         (晶振)   |
     +-||--+          |           |          +-||----+
     22pF 22pF        |           |         22pF  22pF
     |    |           |           |         |     |
     GND  GND         |           |        GND   GND
                       |           |
     +--10uF--+       |           |
     |        |       |   P1.0---+-------> 数码管 a
    RST       |       |   P1.1---+-------> 数码管 b
     +--10K---+VCC    |   P1.2---+-------> 数码管 c
     |    (复位)      |   P1.3---+-------> 数码管 d
     GND              |   P1.4---+-------> 数码管 e
                       |   P1.5---+-------> 数码管 f
                       |   P1.6---+-------> 数码管 g
                       |           |
                       |   P2.0---+-------> 数码管十位 公共端（低有效）
                       |   P2.1---+-------> 数码管个位 公共端（低有效）
                       |           |
                       |   P3.2---+-------> K2 按键 ---> GND
                       |   P3.3---+-------> K1 按键 ---> GND
                       |           |
                       |   P3.7---+-------> SPWM 输出
                       |           |
                       +-----------+

    数码管（两位一体共阳）:
    ┌─────────────────────────────┐
    │  ┌───┐         ┌───┐       │
    │  │ 1 │ (十位)  │ 0 │(个位) │   显示 "10" 表示 10Hz
    │  └─┬─┘         └─┬─┘       │
    │    │              │         │
    │   COM1           COM2       │
    │   (P2.0)        (P2.1)     │
    └─────────────────────────────┘
    段选 a~g 并联接 P1.0~P1.6
    每位公共端分别接 P2.0 / P2.1
```

### SPWM 输出驱动（可选）

当负载电流较大时，P3.7 输出端建议增加驱动电路：

```
方案一：三极管驱动（小功率负载）

    P3.7 ──[1K]──┬── R
                 │
                B │ NPN (如 8050)
                E │── GND
                 │
                C │── 负载 ── VCC

方案二：光耦隔离（需隔离的功率负载）

    P3.7 ──[330]── LED(光耦输入) ── GND
                   光耦输出端驱动后级功率器件
```

## 软件架构

```
main.c            主程序入口，全局频率变量，主循环调度
├── spwm.c/h      SPWM 核心：100 点正弦查找表 + Timer0 双相中断
├── display.c/h   数码管动态扫描显示（P1 段选 + P2 位选）
├── key.c/h       按键扫描、消抖、频率加减
└── delay.c/h     毫秒/微秒级软件延时
```

## 编译说明

1. 使用 Keil uVision 打开 `test.uvproj`
2. 确认 Target Options 中 Xtal 设置为 **11.0592MHz**
3. 编译（Build），生成 HEX 文件位于 `Objects/` 目录
4. 使用下载器将 HEX 烧录到 ST89C52
